## Features

- **Real-time Precision**: Powered by `OpenWeatherMap` for accurate global weather data.
- **Dynamic Visuals**: Adaptive background gradients and glassmorphism UI that change with the weather conditions.
- **Smart Forecast**: 5-day predictive outlook with intuitive visualizations.
- **Geolocation**: Instant local weather data via browser geolocation.
- **Rich UX**: 
  - Custom colored vector weather icons.
  - Interactive search with validation.
  - High-contrast typography for readability on any background.
  - Fully responsive layout for mobile and desktop.
- **Tech Stack**: React 18, Tailwind CSS, Lucide React, Axios, Vite.

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Configure your API key in `constants.ts`.
4.  Run the development server:
    ```bash
    npm run dev
    ```
